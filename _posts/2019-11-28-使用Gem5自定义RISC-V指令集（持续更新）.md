## 环境搭建
### Gem5 编译和示例运行
Gem5的编译，需要依赖一些库文件，在ubuntu上面可以执行下条命令完成安装：
```
sudo apt-get install git build-essential python-dev python scons zlib1g zlib1g-dev m4 libprotobuf-dev protobuf-compiler libprotoc-dev libgoogle-perftools-dev
```
